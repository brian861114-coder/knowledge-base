# GitHub Pages Deploy Folder

This `docs/` folder is a self-contained static build for GitHub Pages.

Publish settings:

- Branch: `main`
- Folder: `/docs`

Entry point:

- `docs/index.html`
- `docs/materials-science-engineering/index.html`

Static data files expected by the app:

- `docs/physics_graph.json`
- `docs/physics_note_details.json`
- `docs/materials-science-engineering/graph.json`
- `docs/materials-science-engineering/note_details.json`
