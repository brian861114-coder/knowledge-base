# Content Workflow

## Recommended Editing Order

1. Expand or refine map pages.
2. Add missing principles that explain major causal links.
3. Add concept notes for structure and material classes.
4. Add property notes that capture decision variables.
5. Add process and characterization notes.
6. Add failure and application clusters last, after the core network is stable.

## Minimum Note Standard

Every note should have:

- `title`
- `type`
- `summary`
- `domain`
- `taxonomy_domain`

Every note should also contain explanatory prose, not just bare links.

## Link Discipline

Prefer links that answer one of these questions:

- what causes this
- what measures this
- what changes this
- what depends on this
- where this is used
- what can go wrong

If a note cannot explain why a link exists, the link is probably noise.
