# Page Types

This skill uses page types instead of loose note categories.

## `law`

Use for physical laws, theorems, conservation laws, and canonical equations.

Examples:

- 牛頓第二定律
- 庫侖定律
- 高斯定律
- 熱力學第一定律

Required emphasis:

- applicability
- mathematical form
- physical intuition
- derivation
- experiments
- derived results

## `concept`

Use for abstract physical ideas.

Examples:

- 力
- 場
- 能量
- 熵

Required emphasis:

- definition
- meaning
- intuition
- representative laws

## `quantity`

Use for measurable quantities, constants, and formal physical variables.

Examples:

- 質量
- 加速度
- 動量
- 電荷

Required emphasis:

- symbol
- unit
- dimension
- measurement
- appearance in laws

## `experiment`

Use for classical experiments, validation methods, and measurement setups.

Examples:

- 密立根油滴實驗
- 楊氏雙縫實驗
- 卡文迪西實驗

Required emphasis:

- setup
- observables
- results
- supported laws

## `mathematical_tool`

Use for mathematical tools needed to understand or formalize physics.

Examples:

- 向量分解
- 微分
- 積分
- 散度

Required emphasis:

- definition
- geometric meaning
- why physics uses it
- where it appears

## `map`

Use for navigation and learning-path pages.

Examples:

- 力學總覽
- 電磁學學習路徑
- 熱學概念地圖

Required emphasis:

- scope
- learning order
- included pages
