# Physics Structure

This demo assumes a university-level general physics encyclopedia.

## Reader-Facing Top Domains

- 力學
- 熱學
- 電磁學
- 波動與光學
- 近代物理

## System-Facing Page Types

- `law`
- `concept`
- `quantity`
- `experiment`
- `mathematical_tool`
- `map`

## Recommended Build Order

1. Build the first `law` pages
2. Add supporting `concept` and `quantity` pages
3. Add representative `experiment` pages
4. Add `mathematical_tool` pages where formal support matters
5. Build `map` pages for domain navigation
