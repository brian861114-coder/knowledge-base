# 電磁學總覽

## 地圖摘要
電磁學最大的難點，不是公式多，而是層級很多：從粒子受力、到場、到通量、到感應、到波，全都接在一起。

## 範圍說明
本頁整理電磁學主幹，幫你看清楚靜電學、電路、磁學、感應與電磁波其實是同一個理論體系的不同切面。

## 主要主題
- 靜電主線：[[電荷]] -> [[庫侖定律]] -> [[電場]] -> [[電位]] -> [[電位能]]
- 場論主線：[[電場]] -> [[通量]] -> [[高斯定律]]
- 電路主線：[[電位]] -> [[歐姆定律]]
- 感應主線：[[磁場]] -> [[磁通量]] -> [[法拉第定律]] -> [[麥克斯威方程組]] -> [[電磁波]]

## 延伸方向
- 電磁學最容易學壞的方式，是永遠停在單一點電荷公式。
- 真正的進步點，是從粒子受力觀點切換到場與通量，再走到感應與波。


## 第二波電路與磁學骨架
- 電路主線：[[電位差]] -> [[電動勢]] -> [[電流]] -> [[電阻]] -> [[直流電路]] -> [[歐姆定律]]
- 磁場來源主線：[[電流]] -> [[畢奧-沙伐定律]] / [[安培定律]] -> [[磁場]]
- 感應主線：[[通量]] -> [[磁通量]] -> [[法拉第定律]] -> [[楞次定律]] -> [[電感]] -> [[交流電]]
- 統整主線：[[高斯定律]] + [[法拉第定律]] + [[畢奧-沙伐定律]] / [[安培定律]] -> [[麥克斯威方程組]]

## 物理直覺
電磁學總覽 的用途不是把頁面排成漂亮清單，而是替這個領域建立閱讀順序與概念主幹。先看哪些節點是骨架、哪些是支撐、哪些是應用，理解速度會比逐頁亂跳快得多。

## 歷史背景
這類總覽頁本身不是歷史對象，但它反映了一種後見之明：只有在大量主題已經彼此連接之後，才可能把整個領域重新整理成較清楚的知識地圖。
